# Friends Workspace

Friends Workspace is a full-stack platform for workspace booking, desk management, and amenity reservations.

## Tech Stack
- Backend: Spring Boot
- Frontend: React (Vite)

## Run Locally
1. Backend
```bash
cd backend
mvn spring-boot:run
```
2. Frontend
```bash
cd frontend
npm install
npm run dev
```
